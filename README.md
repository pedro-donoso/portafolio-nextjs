## https://pedro-donoso.vercel.app/

### 1. clonar repositorio:

```
git clone https://github.com/pedro-donoso/portafolio-nextjs.git
```

### 2. instalar dependencias:

```
node -v
npm -v
```

### 3. Instalar dependencias del proyecto:

```
npm install
```

### 4. Verificar

```
ls
```

### 5. Ejecutar:

```
npm run dev
```
